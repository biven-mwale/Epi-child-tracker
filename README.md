# EPI Child Tracker — Android App

Zambia Under-5 Immunisation Tracker for Western Province Health Office, Kaoma District.

## Features
- Register children with Under 5 Card Number (PPDD-FF-YYYY-NNNN format)
- Full Zambia EPI schedule (24 doses across 11 visit groups)
- Includes Malaria (RTS,S), Rotavirus 3, IPV 1 & 2
- PIN security with auto-lock
- Works 100% offline
- Export CSV for DHIS2 or Excel

## Download APK
Go to the **Releases** tab on the right side of this page and download the latest `EPI-Child-Tracker.apk`

## Install on Android
1. Download `EPI-Child-Tracker.apk`
2. Send to phone via WhatsApp or USB
3. On the phone: Settings → Security → Allow unknown sources
4. Open the APK file and tap Install

## Build Status
The APK is built automatically by GitHub Actions every time code is updated.
